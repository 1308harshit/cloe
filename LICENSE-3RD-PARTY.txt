3RD-PARTY LICENSES
==================

Bundled 3rd-Party Libraries
---------------------------

The following 3rd-party libraries are included in the Cloe repository:

Bootstrap
    License: MIT
    License-Source: https://github.com/twbs/bootstrap/blob/v4.0.0-beta.2/LICENSE
    Website: http://getbootstrap.com
    Source: engine/webui/bootstrap.min.css
    Description:
        We use Bootstrap for our minimal web interface.
        Bundled because Cloe should be able to run offline.
